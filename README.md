# Java 学习计划仓库

这是一个使用 AI 辅助学习的 Java 编程学习仓库，采用苏格拉底式教学方法，帮助你系统地掌握 Java 编程。

---

**学习目标**: 系统掌握 Java 核心知识，从基础到高级
**学习方法**: AI 引导式学习 + 每日记录 + 进度追踪

## 项目结构

```
/sessions/                    # 每日学习记录
  /YYYY-MM-DD/               # 按日期组织的学习笔记
  SESSION-TEMPLATE.md        # 学习记录模板

/progress/                    # 学习进度追踪
  java-study-tracker.md      # 综合进度追踪文件

/code-examples/              # 代码示例
  /basics/                   # 基础语法示例
  /oop/                      # 面向对象示例
  /collections/              # 集合框架示例
  /concurrency/              # 并发编程示例
  /advanced/                 # 高级特性示例

/resources/                  # 学习资源链接

CLAUDE.md                    # AI 导师指令
README.md                    # 本文件
```

## 如何使用

### 每日学习

1. 打开 Kiro/Claude 在此仓库中
2. 自然地提问 Java 相关问题
3. 回答 AI 提出的理解检查问题
4. 每次学习后，AI 会自动记录：
   - 学到了什么
   - 哪些地方有困难
   - 掌握了哪些知识点
   - 下次需要复习什么

### 复习模式

可以这样请求复习：
- "让我们复习一下我之前不太懂的内容"
- "今天应该重点学什么？"
- "测试一下我的薄弱环节"
- "展示我的学习进度"

### 查看进度

查看 `/progress/java-study-tracker.md` 了解：
- 整体学习进度
- 各模块完成情况
- 知识薄弱点
- 优先学习计划

## 学习理念

**引导式学习方法：**
- 对话式、无压力的学习环境
- 基于你已有的知识构建新知识
- 在继续之前确认理解
- 适应你的学习风格
- 注重深度理解，而非死记硬背

## Java 学习路线

### 第一阶段：Java 基础 (2-3周)
- 环境搭建与开发工具
- 基本语法与数据类型
- 运算符与表达式
- 流程控制
- 数组

### 第二阶段：面向对象编程 (3-4周)
- 类与对象
- 封装、继承、多态
- 抽象类与接口
- 内部类
- 枚举

### 第三阶段：核心API (2-3周)
- String 与 StringBuilder
- 包装类
- 日期时间 API
- 异常处理
- 文件 I/O

### 第四阶段：集合框架 (2周)
- List、Set、Map
- 迭代器
- 泛型
- Collections 工具类

### 第五阶段：高级特性 (3-4周)
- 多线程与并发
- Lambda 表达式
- Stream API
- 反射
- 注解

### 第六阶段：实战应用 (持续)
- JDBC 数据库操作
- 网络编程
- 设计模式
- 项目实战

## 开始学习

直接开始和 AI 对话，提出你的第一个 Java 问题。AI 会引导你学习，同时自动追踪你的进度。
